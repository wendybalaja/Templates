Detailed Description

template<typename T, typename C = comp<T>>
class bin_search_simple_set< T, C >

Sorted array implementation of set; supports binary search. Requires instantiation with guaranteed upper bound on number of elements that may be placed in set. Throws overflow if bound is exceeded.

'virtual' on simple_set ensures single copy if multiply inherited
