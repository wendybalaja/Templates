Detailed Description

template<typename T>
class carray_simple_set< T >

Characteristic array implementation of set. Requires instantiation with guaranteed low and one-more-than-high bounds on elements that can be placed in the set. Should compile and run correctly for any element class T that can be cast to int. Throws out_of_bounds exception when appropriate.
