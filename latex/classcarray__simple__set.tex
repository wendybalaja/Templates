class carray\+\_\+simple\+\_\+set$<$ T $>$}

Characteristic array implementation of set. Requires instantiation with guaranteed low and one-\/more-\/than-\/high bounds on elements that can be placed in the set. Should compile and run correctly for any element class T that can be cast to int. Throws \mbox{\hyperlink{classout__of__bounds}{out\+\_\+of\+\_\+bounds}} exception when appropriate.
