\subsubsection*{template$<$typename T, typename F = cast\+\_\+to\+\_\+int$<$\+T$>$$>$\newline
class hashed\+\_\+simple\+\_\+set$<$ T, F $>$}

Hash table implementation of set. Requires instantiation with guaranteed upper bound on number of elements that may be placed in set. Throws overflow if bound is exceeded. Can be instantiated without second generic parameter if element type can be cast to int; otherwise requires hash function object.
